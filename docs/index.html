  LLF-Bench : Benchmark for Interactive Learning from Language Feedback

A collection of 8 diverse learning problems.

LLF-Bench includes diverse environments such as navigation, user recommendation, robotic manipulation domain, and poem generation.

Exposes prompt hacking

LLF-Bench generates and randomizes verbalization making it hard for LLM-based agents to prompt hack to any given environment.

Requires learning

Some environments in LLF-Bench require learning no matter how smart the LLM, or any base foundation model is. This provides a way to evaluate learning in these agents.

What is LLF Bench? LLF-Bench (Learning from Language Feedback Benchmark; pronounced as “elf-bench”), is a new benchmark to evaluate the ability of AI agents to interactively learn from language feedback. The agent interacts with an environment in LLF-Bench, takes action, and get language feedback instead of rewards or expert actions. LLF-Bench consists of 8 diverse benchmarks

                                    git clone https://github.com/microsoft/LLF-Bench.git

                                    conda create -n LLF-Bench python=3.8 -y

                                    conda activate LLF-Bench

How is it different from RL? Reinforcement learning (RL) is another commonly used interactive learning setting. They key difference is that in RL, the agent is trained using rewards, whereas in LLF (the paradigm upon which LLF-Bench) is based, uses language feedback instead of rewards.

Why language feedback? Language feedback has two main advantages of rewards and expert actions - the two most commonly used feedbacks. Firstly, unlike rewards, language feedback is very expressive and consequently can pack a lot more information which can help the agent train faster, and unlike actions, language feedback can be more easily provided by non-expert humans. Secondly, language feedback is closer to how humans learn, and this makes it more natural for many settings.

Can LLF-Bench be used to evaluate LLMs? Yes! In fact, that is one of the main intensions -- to provide robust evaluation for LLM Agents.
